Структура архива memorial-landing:

1. index.html
   Главная страница лендинга.

2. assets/css/style.css
   Все стили проекта.

3. assets/js/script.js
   Скрипты: мобильное меню и модальное окно галереи.

4. assets/images/clouds-bg.png
   Фон с облаками, приложенный пользователем.

5. assets/images/hero-monument.png
   Главное изображение справа в первом экране, приложенное пользователем.

6. assets/placeholders/*.svg
   Обязательные заглушки под фотогалерею и аксессуары.
   Их можно заменить на реальные изображения без изменения структуры HTML.

7. assets/images/reference-layout.png
   Референс-макет, приложенный пользователем, оставлен в архиве как вспомогательный файл.

Что нужно заменить перед публикацией:
- телефон;
- ссылку WhatsApp;
- город / регион;
- политику конфиденциальности;
- реальные цены вместо "от … ₽";
- заглушки в галерее и блоке аксессуаров.
